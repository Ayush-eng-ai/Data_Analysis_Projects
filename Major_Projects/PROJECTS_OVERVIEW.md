## 📊 How to View Dashboards

### Power BI Files (.pbix)
- Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
- Open the `.pbix` files to view interactive dashboards
- Requires Windows or Power BI Service

### Excel Files (.xlsx, .xlsm)
- Open with Microsoft Excel or compatible software
- View dashboard sheets and raw data
- Macros may require Excel to function properly

### CSV Data
- Open with any text editor or Excel
- View raw data for SQL queries or further analysis

### GIF Files
- View in any image viewer or browser
- Preview dashboard animations and functionality
