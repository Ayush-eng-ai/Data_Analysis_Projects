# 🌦️ Weather with API Dashboard Creation

## 📌 Project Overview

The **Weather with API Dashboard Creation** project is a dynamic and interactive weather analytics dashboard developed using real-time weather API data and Power BI visualization techniques.

This project fetches live weather information through an API and transforms raw weather data into visually appealing and meaningful insights using modern dashboard design principles.

The dashboard provides:
- Real-time weather updates
- Forecast analysis
- Air Quality Index monitoring
- Sunrise & Sunset information
- Rain probability analysis
- Interactive weather visualization

---

# 🎯 Project Objectives

- Fetch live weather data using API integration
- Build a dynamic and interactive dashboard
- Visualize real-time weather insights
- Monitor air quality conditions
- Display forecast weather trends
- Create a modern UI dashboard experience
