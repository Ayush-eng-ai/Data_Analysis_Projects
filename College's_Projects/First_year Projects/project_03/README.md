# Credit Rating Dataset Statistical Analysis

## 📌 Project Overview

This project was completed as part of my college Extra Activity for Statistics and Data Analysis.

The objective of this project was to analyze a synthetically generated Credit Rating Dataset of corporate bonds using statistical concepts such as:
- Scatterplots
- Covariance
- Mean
- Standard Deviation
- Chebyshev’s Inequality

The dataset contains scaled financial variables and credit rating information for different corporate bonds.

This project helped me understand:
- Relationships between numerical variables
- Statistical variability
- Data distribution behavior
- Probability bounds using Chebyshev’s theorem

---

## 🎯 Objective

The main objective of this project was to:
- Perform statistical analysis on financial data
- Visualize relationships between variables
- Calculate covariance between selected numerical variables
- Analyze mean and standard deviation
- Apply Chebyshev’s inequality to estimate probability bounds

---

## 📂 Dataset Information

The dataset used in this project is a synthetically generated and scaled Credit Rating Dataset.

Each row represents:
- A corporate bond
- Financial indicators
- Credit rating information

### Selected Numerical Variables
The following variables were selected for analysis:

- CurrentRatio
- QuickRatio
- ROCE

These variables were used for:
- Scatterplot analysis
- Covariance calculation
- Mean & standard deviation analysis
- Chebyshev’s inequality

---

## 📊 Scatterplot Analysis

Scatterplots were created to visualize relationships between:
- CurrentRatio vs QuickRatio
- QuickRatio vs ROCE
- CurrentRatio vs ROCE

### Observations
- Some variable pairs showed weak positive relationships
- Certain pairs displayed scattered distributions indicating low correlation
- The plots helped identify trends and variability in financial ratios

Scatterplots are useful for understanding:
- Correlation patterns
- Clusters
- Outliers
- Direction of relationships

---

## 📈 Covariance Analysis

Covariance was calculated for:
- Cov(CurrentRatio, QuickRatio)
- Cov(QuickRatio, ROCE)
- Cov(CurrentRatio, ROCE)

### Interpretation
- Positive covariance indicates variables tend to increase together
- Negative covariance indicates inverse movement between variables
- Covariance values near zero suggest weak linear relationships

This analysis helped in understanding how financial indicators move relative to each other.

---

## 📐 Mean and Standard Deviation

Mean and standard deviation were calculated for the selected numerical variables.

### Mean
The mean represents the average value of the variable.

### Standard Deviation
Standard deviation measures how spread out the values are from the mean.

### Interpretation
- Lower standard deviation indicates more stable values
- Higher standard deviation indicates higher variability
- Mean values helped identify the central tendency of the data

---

## 🧮 Chebyshev’s Inequality

Chebyshev’s inequality was applied to estimate probability bounds for the selected variables.

The theorem helps determine:
- Lower bound probability
- Upper bound probability
- Range in which most values are expected to lie

### Example Interpretation
Using the calculated mean and standard deviation:
- A large percentage of observations lie within a certain range from the mean
- Variables with larger variability produce wider probability bounds

This theorem is useful when:
- Data distribution is unknown
- Estimating variability and uncertainty
- Understanding spread in financial datasets
